Веб-приложение для управления кадровым учётом организации. Позволяет HR-специалистам просматривать, добавлять, редактировать и увольнять сотрудников.

Перед запуском убедитесь, что установлены:
Node.js (версия 18 или выше)

Инструкция по запуску

Шаг 1. Клонируйте репозиторий
Скачайте и распакуйте либо через команду: git clone <URL-репозитория>
cd areal

Шаг 2. Установите зависимости
npm install

Шаг 3. Запустите сервер
node server.js

Либо

npm start

Структура проекта:
areal/
├── db.json                   # База данных
├── package.json              # Зависимости проекта
├── package-lock.json         # Зафиксированные версии зависимостей
├── README.md                 # Документация
├── server.js                 # Точка входа (Express сервер)
│ 
└── public/
├── index.html       # HTML-разметка
├── style.css        # Стили
└── script.js        # Фронтенд-логика
